 Pawfect Care – Pet Service Platform

 Project Description

Pawfect Care is a web-based application that helps users manage pet services like grooming, booking, and product purchases. It provides an easy and user-friendly interface for pet owners.

##  Features

* User Login & Registration
* Admin Dashboard
* Book Pet Grooming Services
* View & Buy Pet Products
* Secure Authentication (JWT)
* Responsive UI

##  Tech Stack

### Frontend:

* React (Vite)
* Tailwind CSS / Bootstrap

### Backend:

* Node.js
* Express.js
* MongoDB

## Project Structure

* Frontend → React (UI)
* Backend → Express Server
* Database → MongoDB

##  Installation

### 1. Clone repo

```bash
git clone <your-repo-link>
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run backend

```bash
npm start
```

### 4. Run frontend

```bash
npm run dev
```

##  Environment Variables

Create a `.env` file and add:
JWT_SECRET=your_secret_key
